# GitHub Yearly Activity Card (032)

This project generates a single stats card image for one GitHub username and outputs.

## Features
- Fetches yearly GitHub activity (commits, PRs, issues, events, streak)
- Creates a profile card image with the generated stats
- Provides copy/download actions from the UI

## Usage
1. Open `index.html` in a browser
2. Enter a GitHub username
3. Click `Generate Image`
4. Copy the generated Markdown and paste it where needed

## Files
- `index.html`: UI
- `style.css`: Layout and styling
- `script.js`: Bootstrap
- `modules/`: App logic, API, rendering, and card generation
