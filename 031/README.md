# 031 Voice Court (STT/TTS)

사용자 발화(STT)로 사건을 접수하고, 변호/검사 공방, 배심원 키워드 반영, 최종 평결(유죄/무죄)까지 진행하는 게임입니다.

## GitHub Pages

- https://dusunax.github.io/exercises/031/

## 핵심 기능

- 사건 접수 STT
  - 중앙 `CASE FILE` 버튼으로 사건 주제 접수
  - 판사 자막 연출: "그럼, 재판을 시작합니다." → 1초 후 "이번 사건은..."
- 공방 진행
  - 변호/검사/판사 턴 기반 스크립트 TTS 진행
  - 캐릭터별 말풍선, 스프라이트 상태 전환
  - 자동 로그 누적
- 단일 게이지 전투 시스템
  - 하나의 긴 게이지를 변호/검사가 공유
  - 클릭/음성 키워드/STT 결과로 증감
  - 50:50(동점)은 유죄 판정
- 턴 데미지 연출
  - 변호/검사 턴마다 상대에게 `1~10` 데미지
  - 데미지 텍스트 머리 위 팝업
  - `8~10`은 `CRITICAL` 이펙트 + 타격 연출
- 배심원 키워드 감지(STT)
  - 개정 중 자동 STT 감시
  - 긍정/부정 키워드 입력 시 1) 중앙에 키워드 부스터 이펙트, 2) 게이지 반영
- 평결 STT
  - 배심원 평결은 `10초 내` 유죄/무죄 발화
  - UI 상단에 시간 제한 힌트 고정 표기
- 판결/결과 연출
  - 전면 유죄/무죄 버스트 표시(5초)
  - 판결문은 중앙 팝업으로 출력(재심/닫기 버튼)
  - 점수 표시: 유죄=검사 게이지*100, 무죄=변호 게이지*100
- 사운드
  - 우측 상단 스피커 아이콘으로 ON/OFF
  - 상태별 8bit BGM (개정 전/개정 중/긴박/판결/완료)

## 게임 상태 흐름

- `idle`
- `listening_case`
- `case_ready`
- `court_in_session`
- `listening_verdict`
- `verdict_ready`
- `verdict_reading`
- `done`
- `error`

## UI 요약

- 상단 중앙 로고 + 도트 격투게임형 HUD
- 중앙 법정 무대(좌 변호 / 상 판사 / 우 검사)
- 말풍선 + 도트 스프라이트 + 상태별 표정/포즈
- 하단 증언 기록 / 판결문 카드
