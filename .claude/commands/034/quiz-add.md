# quiz-add

새 퀴즈를 추가한다.

## 입력
- `$1` : 카테고리
- `$2` : 난이도 (`easy|medium|hard`)

## 실행 규칙
1. 대상 카테고리(`$1`)와 난이도(`$2`)를 기반으로 새 문항 형식을 아래와 동일하게 준비한다.
2. 다음 위치에 항목을 추가한다: `src/data/questionBank.js`의 `const QUESTIONS = [` 배열 안, 기존 형식(쉼표 포함)을 유지.
3. ID는 기존 마지막 ID 다음 번호로 지정한다.
4. 형식은 아래와 동일해야 한다.

```javascript
{
  id: <newId>,
  category: "$1",
  difficulty: "$2",
  question: "문항 본문",
  options: ["선택지1", "선택지2", "선택지3", "선택지4"],
  correctAnswer: <0|1|2|3>,
  explanation: "해설"
}
```

## 필수 검증 가이드(반드시 통과)
- 질문은 한글 문장으로 명확하고 단정적으로 작성한다.
- 정답은 options 인덱스(0~3) 정확히 지정.
- `option`은 4개, 중복 없이 의미를 구분.
- 출제 의도와 해설이 일치해야 함.
- 모호 표현 점검 규칙(`가장`, `최초`, `최대`)을 위반하지 않거나,
  아래 기준을 반드시 포함해 보강한다:
  - 비교 기준(범위/시점/기관)
  - 수치/단위 또는 근거 출처
  - 한정 대상(세계/국내/카테고리)

## 추가 후 체크리스트
1. `/.claude/commands/quiz-validate.md`로 새 문항 검증
2. `/.claude/commands/quiz-range.md`로 최근 구간 분포 확인
3. 기존 문항과 동일한 JSON 스타일을 유지했는지 확인
