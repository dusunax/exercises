# teacher-compare

`teacher_scores.json`을 읽어 학생들의 성적을 카테고리별·전체별로 비교한다.

## 입력
- `$ARGUMENTS` : 비교 범위(선택)
  - 비어 있거나 `all` : 전체 성적 비교
  - 카테고리명 (`여행 | 음식 | 일상 | 미술/예술`) : 해당 카테고리만 비교
  - 학생 이름 2개 이상 : 지정 학생들만 1:1 비교 (예: `홍길동 김철수`)

## 데이터 파일
- `034/teacher_scores.json`

## 실행 규칙

### 1단계 — 데이터 로드
`teacher_scores.json`을 읽는다.
- 파일이 없거나 학생이 0명이면 → "성적 데이터 없음" 보고 후 종료.

### 2단계 — 학생별 대표 점수 계산
각 학생의 최고 점수(카테고리·전체 구분)를 계산한다.
- 동일 카테고리 기록이 여러 개이면 최고 점수를 대표값으로 사용.
- 전체(`전체` 카테고리) 점수가 없으면 카테고리 점수 평균으로 대체.

### 3단계 — 상대 등급 산정
전체 학생의 대표 점수를 기준으로 상대 등급을 부여한다:
- 상위 20% 이내 : **A**
- 상위 21~40% : **B**
- 상위 41~70% : **C**
- 하위 30% : **D**

### 4단계 — 카테고리별 비교표 출력
`$ARGUMENTS`로 카테고리가 지정된 경우 해당 카테고리만, 아니면 전체를 출력한다.

## 출력 템플릿

```
=== 034 퀴즈 학생 성적 비교 ===
비교 범위 : [전체 / 카테고리명 / 지정 학생 목록]
기준 날짜 : YYYY-MM-DD

[전체 성적 비교]
순위 | 학생    | 전체   | 여행   | 음식   | 일상   | 미술/예술 | 등급
-----|---------|--------|--------|--------|--------|-----------|-----
  1  | 홍길동  |   92점 |   95점 |   88점 |   90점 |     95점  |  A
  2  | 김철수  |   78점 |   80점 |   75점 |   70점 |     87점  |  B
...

[카테고리별 1위]
여행: 홍길동 95점 | 음식: 이영희 90점 | 일상: ... | 미술/예술: ...

[통계]
평균 점수 : N점
최고 점수 : N점 (학생명)
최저 점수 : N점 (학생명)
표준편차  : N점
```

## 규칙
- 기록이 없는 카테고리는 `-`로 표시한다.
- 순위는 전체 점수 기준으로 정렬한다.
- 동점자는 같은 순위를 부여하고 이름 가나다순으로 표시한다.
