# quiz-range

지정한 ID 구간(`$1`~`$2`) 문제를 중심으로 난이도와 정답 분포를 확인한다.

## 입력
- `$1` : 시작 문제 번호 (필수)
- `$2` : 종료 문제 번호 (필수)

## 대상 파일
- `src/data/questionBank.js`

## 실행 규칙
1. `$1`, `$2`를 숫자로 파싱한다.
2. `id` 기준으로 `id >= $1 && id <= $2`인 문제만 추출한다.
   - 범위를 벗어나거나 존재하지 않는 ID가 포함되면 경고한다.
3. 추출 대상이 없으면 즉시 종료한다.
4. 분석 항목
   - 난이도 분포: `easy / medium / hard` 개수
   - 정답 분포: `correctAnswer` 값(0~3) 개수
   - 옵션 균형: 각 보기는 전체 정답 분포에서 몇 번 정답인지
   - 카테고리 분포(선택)
5. 불균형 경고 규칙(예시)
   - 난이도 한 쪽이 전체의 70% 초과
   - 정답 인덱스 편중(특정 인덱스 50% 초과)

## 출력 템플릿
- 범위: `$1`~`$2`
- 총 문항 수
- 난이도 분포 표
- 정답 인덱스 분포 표
- 카테고리 분포(있으면)
- 경고/리스크
- 개선 제안
