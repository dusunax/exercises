# teacher-insights

`teacher_scores.json`의 전체 성적을 분석해 학급 취약점과 개선 포인트를 도출한다.

## 입력
- `$ARGUMENTS` : 분석 대상(선택)
  - 비어 있거나 `all` : 전체 학생 분석
  - 학생 이름 : 해당 학생 개인 분석
  - 카테고리명 : 해당 카테고리 심화 분석

## 데이터 파일
- `034/teacher_scores.json`
- `034/src/data/questionBank.js` (카테고리·난이도 분포 참조)

## 실행 규칙

### 1단계 — 학급 전체 분석 (기본)
- 학생 수, 총 기록 수
- 평균·최고·최저 점수, 표준편차
- 점수대 분포: 90+, 80~89, 70~79, 60~69, ~59

### 2단계 — 카테고리별 취약 분석
각 카테고리의 학급 평균 점수를 계산하고 내림차순 정렬한다.
- 학급 평균이 가장 낮은 카테고리 → "학급 취약 카테고리" 지정
- 개인 평균이 학급 평균보다 10점 이상 낮은 학생 → "개인 취약" 플래그

### 3단계 — 성장 추이 분석
동일 학생이 같은 카테고리에 2회 이상 기록이 있으면 추이를 분석한다:
- 점수 상승 → ↑ 표시
- 점수 하락 → ↓ 표시
- 유지 → → 표시

### 4단계 — 개선 권고
- 학급 전체: 취약 카테고리 집중 연습 권고
- 개인: 취약 학생 목록과 해당 카테고리 추가 연습 권고

## 출력 템플릿

```
=== 034 퀴즈 학급 인사이트 분석 ===
분석 대상  : [전체 / 학생명 / 카테고리]
분석 일시  : YYYY-MM-DD

[학급 요약]
학생 수   : N명  |  총 기록 수: N개
평균 점수 : N점  |  표준편차: N점
최고 점수 : N점 (학생명)  |  최저 점수: N점 (학생명)

[점수 분포]
90+ : ██████ N명 (X%)
80~ : ████   N명 (X%)
70~ : ███    N명 (X%)
60~ : ██     N명 (X%)
~59 : █      N명 (X%)

[카테고리별 학급 평균]
여행: N점 | 음식: N점 | 일상: N점 | 미술/예술: N점
⚠ 취약 카테고리: [카테고리명] (학급 평균 N점)

[개인 취약 현황]
학생명  | 취약 카테고리      | 학급 평균 대비
--------|--------------------|--------------
홍길동  | 미술/예술          | -15점
...

[성장 추이]
학생명  | 카테고리 | 1회→최근   | 추이
--------|----------|------------|------
...

[권고 사항]
학급: ...
개인: ...
```

## 규칙
- 기록이 1개뿐인 학생은 추이 분석 행에서 제외한다.
- 점수 분포 막대는 최대 8글자로 비례 표시한다.
